Presentacion y ejemplos para la charla de Febrero/2016 del meetup de Python Madrid

En esta charla se expusieron diferentes herramientas provenientes de la programación funcional y/o que se usan en python para este estilo de programación.

Temas expuestos

  • map
  • filter
  • reduce
  • map and filter implemented with reduce
  • List comprehensions
  • Lazy evaluation (through generators and corutines)
  • Function currying and uncurrying
  • Function composition
  • Final example
